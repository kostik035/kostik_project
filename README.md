# kostik_project

Репозиторий, содержащий платформу для фрилансеров (маркетплейс услуг), созданную на Flask

**Приложение позволяет:**

* Просматривать услуги, которые предлагают фрилансеры
* Регистрироваться как покупатель или как продавец (фрилансер)
* Создавать услуги (гиги) (только для продавцов)
* Заказывать услуги (только для покупателей)
* Управлять заказами (принимать, отклонять, запрашивать доработку)
* Оставлять отзывы и оценивать выполненную работу

**Особенности:**

* База данных создана на SQLite
* По умолчанию создается 1 тестовый продавец, 1 тестовый покупатель и 1 пример услуги
* При регистрации, пароль хэшируется с помощью Werkzeug и недоступен для просмотра, так как его нельзя расшифровать
* Удобная главная страница. Каждая карточка услуги содержит название, описание, цену и имя продавца. При нажатии на карточку показывает полную информацию об услуге
* Рейтинг продавцов рассчитывается на основе отзывов покупателей (Сумма звёзд делённая на количество отзывов)
* Уровни продавцов (Новичок, Серебро, Золото) автоматически присваиваются на основе количества завершённых заказов и рейтинга
* Быстрый поиск услуг по названию или описанию
* Четкое разделение: app.py (контроллеры), models.py (модели), templates/ (представления)

**Структура проекта:**
